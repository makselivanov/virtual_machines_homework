## Тесты

Чтобы проверить валидность 
интепретатора на маленьких тестах

```shell
make regression
```

Чтобы проверить все сгенерированные тесты, 
запустите

```shell
make regression-all
```

## Производительность

```shell
make perfomance
```
Сравнит время выполнения встроенного интепретатора 
ламы версии 1.10 и текущего
